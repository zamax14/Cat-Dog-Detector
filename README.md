# Cat-Dog-Detector

## Objetivo
-   Entender que es la IA y sus límites
-   Entrenar o “crear” un modelo de IA para clasificar imágenes de perros y gatos
-   Desplegarlo en una APP móvil
-   Probar la app con imágenes nunca vistas
